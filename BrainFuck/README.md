# esolang/BrainFuck
BrainFuck 的 C 语言和 python 实现

## 编译
```shell
gcc -o BrainFuck BrainFuck.c
```

# 用法
1. 帮助(这玩意还需要帮助?):
```shell
./BrainFuck --help
python BrainFuck.py --help
```

2. 查询版本号(永远1.0):
```shell
./BrainFuck --version
python BrainFuck --version
```

3. 运行(一个文件):
```shell
./BrainFuck example.fb
python BrainFuck.py example.fb
```
